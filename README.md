# Mapa de Mesas

Projeto desenvolvido em HTML e CSS simulando um mapa de mesas de restaurante,
com status visual para mesas livres e ocupadas.

## Tecnologias utilizadas
- HTML5
- CSS3

## Funcionalidades
- Estrutura semântica com table, thead, tbody e tfoot
- Estilização profissional de tabelas
- Hover interativo nas linhas
- Diferenciação visual de status (livre / ocupado)

## Objetivo
Projeto criado para estudo e prática de HTML e CSS.
